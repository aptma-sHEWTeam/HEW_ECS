詳解

前方宣言

エンティティ作成を簡単にするビルダーパターンクラス

メソッドチェーンを使って、複数のコンポーネントを持つエンティティを 簡潔に作成できます。Worldクラスと連携して動作します。

使用例:
Entity player = world.Create()
.With<Transform>(DirectX::XMFLOAT3{0, 0, 0})
.With<MeshRenderer>(DirectX::XMFLOAT3{0, 1, 0}) // 緑色
.With<Rotator>(45.0f)
.Build();
