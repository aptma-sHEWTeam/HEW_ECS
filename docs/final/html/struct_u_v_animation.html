UVスクロールアニメーション（テクスチャ移動）コンポーネント

テクスチャをスクロールさせて流れているように見せます。 用途: 流れる水、回るタイヤ、ベルトコンベア等

使用例（横にスクロール）:
UVAnimation uv;
uv.scrollSpeed = DirectX::XMFLOAT2{0.5f, 0.0f}; // 毎秒0.5横に移動
world.Add<UVAnimation>(entity, uv);
